# DBMS for AI Engineers

## Overview

This repository documents my journey of learning **Database Management Systems (DBMS)** with a focus on practical concepts useful for **AI and Machine Learning engineers**.

In real-world machine learning systems, models rely heavily on **large datasets stored in databases**. Extracting, transforming, and managing this data efficiently requires a strong understanding of **DBMS and SQL**.

This repository combines:

* Core DBMS concepts
* SQL query practice
* Database design fundamentals
* Data extraction techniques for machine learning

The goal is to build a **solid database foundation while connecting it to real ML workflows**.

---

## Why DBMS is Important for AI Engineers

Before training any machine learning model, data typically goes through this pipeline:

Database → SQL Queries → Data Processing → Feature Engineering → ML Model

Understanding DBMS helps in:

* Efficiently extracting training datasets
* Handling large-scale data
* Designing structured datasets
* Optimizing data retrieval performance
